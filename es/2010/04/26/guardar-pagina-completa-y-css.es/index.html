Guardar página completa y CSS

Publicado por nohorbee el 26 de Abril de 2010
Con el avance de las técnicas de diseño de sitios web (escritura de HTML), desde hace ya mucho tiempo, se considera una buena práctica separar lo más posible, el contenido a desplegar, de la manera en que es desplegado. Por este motivo, quienes diseñan y programan (si es que un sitio web se programa), realizan CSS(s) cada vez más completos, en donde incluyen imagenes como fondos, o incluso como objetos de las páginas (lo que comúnmente se hacía simplemente con el tag img). Estas buenas prácticas tienen como efecto colateral, que al momento de querer utilizar la funcionalidad de los browsers "GUARDAR PÁGINA COMPLETA", muchas imágenes no se vean en la página descargada. Esto se debe a que los browsers, descargan las imagenes encontradas en tags HTML y no referenciadas desde CSS.
Para solucionar este problema, existe al menos 1 plug-in de firefox, que encontré simplemente buscando "Save page as fails when images in css" en google. Aquí dejo el link en caso de que este sea el problema

https://addons.mozilla.org/en-US/firefox/addon/4723

Me gustaría encontrar una extensión para Google Chrome que haga lo mismo.
